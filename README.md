# fusion-360-ball-bearing

A parametric script for Autodesk Fusion 360 to model a complete ball bearing assembly consisting of:
* Outer Housing (Ring)
* Inner Housing (Ring)
* Separator (with dynamic hole count optimization)
* Cork / Cap Plug (with screwdriver slot)

## How to Use

1. Open **Autodesk Fusion 360**.
2. Go to the **Utilities** tab on the main toolbar.
3. Select **Add-ins** (or press `Alt + Shift + C`).
4. Select **Script or Add-in from device** (using the **+** symbol to add a local script).
5. Choose **Script** and select the path to the [ballbearing.py](ballbearing.py) file.
6. Select the added script in the list and click **Run**.
